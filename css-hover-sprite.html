CSS Sprite auf Hover

Anstatt auf Hover ein anderes Bild in einem Element zu laden, kann man die Sprite Technik anwenden, nämlich alle Motive, die erscheinen sollen in einem Bild, welches dann auf Hover unterschiedlich positioniert wird. Vorteil: "die Bilder" sind als ein Bild bereits geladen und es kann zu keiner Ladeverzögerung kommen.

Allerdings erfordert diese Technik natürlich Vorarbeit bei der Bildbearbeitung und Komposition.
